# BaiduFocusAssistant
[Greasy Fork 链接](https://greasyfork.org/zh-CN/scripts/394649-baidu-focus-assistant)

让您更好地专注于百度的搜索结果，配合 **Tampermonkey** 使用。  

目前实现的功能有：

- [x] 模糊右侧“搜索热点”、“相关内容”
- [x] 模糊下方“相关搜索”
